Square Root

1. What is Square Root?

Square root of a number x is a number y such that y² = x (e.g., √16 = 4).

2. Methods to Find Square Root

  • Prime Factorization: Factorize, pair factors (e.g., √36 = 6).
  • Long Division Method: For non-perfect squares.
  • Approximation: For irrationals like √2 ≈ 1.414.

3. Example 1: Prime Factorization

Find √144.

  • 144 = 2⁴ × 3².
  • √144 = 2² × 3 = 12.
  • Answer: 12.

4. Example 2: Long Division

Find √27 (approximate to 2 decimals).

  • Pair digits: 27.00.
  • 5² = 25, 27 – 25 = 2; Adjust: 5.2² = 27.04, 5.1² = 26.01.
  • Answer: ≈ 5.20.

5. Example 3: Application

Area of a square is 169 cm². Find side length.

  • Side = √169 = 13 cm.
  • Answer: 13 cm.

6. Practice Problems

  1. Find √64 using prime factorization.
  2. Find √50 to 2 decimals using long division.
  3. Find side of a square with area 225 cm².
  4. Simplify: √(16 × 25).
  5. If √x = 7, find x.

7. Solutions

  1. 64 = 2⁶; √64 = 8.
  2. √50 ≈ 7.07.
  3. Side = 15 cm.
  4. √(16 × 25) = 20.
  5. x = 49.

8. Tips

  • Check if number is perfect square first.
  • Practice long division for approximations.
  • Use √(a × b) = √a × √b for simplification.
