Number System

1. What is the Number System?

The number system is a way to represent numbers using digits or symbols. It forms the foundation of mathematics, used in counting, calculations, and measurements.

2. Types of Numbers

  • Natural Numbers (N): 1, 2, 3, … (counting numbers).
  • Whole Numbers (W): 0, 1, 2, 3, … (natural numbers + zero).
  • Integers (Z): …, -3, -2, -1, 0, 1, 2, 3, … (whole numbers + negatives).
  • Rational Numbers (Q): Numbers as fractions p/q, where q ≠ 0 (e.g., 1/2, 0.75).
  • Irrational Numbers: Non-repeating, non-terminating decimals (e.g., √2, π).
  • Real Numbers (R): All numbers on the number line (rational + irrational).

3. Number Properties

  • Even Numbers: Divisible by 2 (e.g., 2, 4, -6).
  • Odd Numbers: Not divisible by 2 (e.g., 1, 3, -5).
  • Prime Numbers: Divisible only by 1 and itself (e.g., 2, 3, 5).
  • Composite Numbers: More than two factors (e.g., 4, 6, 9).

4. Example 1: Classify Numbers

Classify: 7, -2, 0.5, √3

  • 7: Natural, whole, integer, rational, real.
  • -2: Integer, rational, real.
  • 0.5: Rational, real.
  • √3: Irrational, real.

5. Example 2: Identify Prime Numbers

List prime numbers between 10 and 20.

  • Check: 11, 13, 15, 17, 19.
  • 15 is divisible by 3 (not prime).
  • Answer: 11, 13, 17, 19.

6. Example 3: Rational or Irrational

Determine if 0.333… is rational.

  • 0.333… = 1/3 (repeating decimal).
  • Answer: Rational.

7. Operations on Number Types

  • Adding integers: (-3) + 5 = 2.
  • Multiplying rationals: (1/2) × (3/4) = 3/8.
  • Approximating irrationals: 7/16 ≈ 7.414.

8. Practice Problems

  1. Classify: -5, 0, 2/3, π.
  2. List prime numbers between 20 and 30.
  3. Is 0.121121112… rational or irrational?
  4. Find: (-7) + 4 – (-2).
  5. Multiply: (2/5) × (15/4).

9. Solutions

  1. -5: Integer, rational, r; 0: Whole, integer, rational, r; 2/3: Rational, r; π: Irrational, real.
  2. 23, 29 (check divisibility).
  3. Non-repeating, non-terminating: Irrational.
  4. (-7) + 4 – (-2) = -7 + 4 + 2 = -1.
  5. (2/5) × (15/4) = 30/20 = 3/2.

10. Tips

  • Memorize number types and their properties.
  • Check divisibility for prime numbers.
  • Practice converting decimals to fractions.
