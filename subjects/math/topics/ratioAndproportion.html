Ratio and Proportion

1. What is a Ratio?

A ratio expresses a comparison between two quantities of the same kind, written as a:b or a/b. Example: 2 liters of milk to 3 liters of water → 2:3. It can also be simplified: 10:15 = 2:3 (divide both by HCF 5).

Antecedent: first term (a); Consequent: second term (b)

2. What is a Proportion?

A proportion states that two ratios are equal: a:b = c:d or a:b::c:d, meaning a/b = c/d :contentReference[oaicite:1]{index=1}.

The means are the 2nd and 3rd terms (b and c); the extremes are the 1st and 4th (a and d).

3. Types of Proportion

  • Direct Proportion: As one increases, the other increases. Expressed as y ∝ x :contentReference[oaicite:2]{index=2}.
  • Inverse Proportion: As one increases, the other decreases. Expressed as y ∝ 1/x :contentReference[oaicite:3]{index=3}.
  • Continued Proportion: Series like a:b = b:c = c:d. Then a:b:c:d is in continued proportion :contentReference[oaicite:4]{index=4}.

4. Properties & Tricks

  • Cross-Multiplication: a/b = c/d → ad = bc :contentReference[oaicite:5]{index=5}.
  • Invertendo: a:b = c:d → b:a = d:c :contentReference[oaicite:6]{index=6}.
  • Alternendo: a:b = c:d → a:c = b:d :contentReference[oaicite:7]{index=7}.
  • Addendo: a:b = c:d → (a+c):(b+d) :contentReference[oaicite:8]{index=8}.
  • Subtrahendo: (a–c):(b–d), etc. :contentReference[oaicite:9]{index=9}.
  • Componendo–Dividendo: (a+b):(a−b) = (c+d):(c−d) :contentReference[oaicite:10]{index=10}.

5. Worked Examples

Example 1: Are 6:8 and 24:32 in proportion?

  • 6/8 = 0.75; 24/32 = 0.75 → Yes, they are proportional. :contentReference[oaicite:11]{index=11}

Example 2: If 3/5 = x/20, find x using cross-multiplication:

  • 3×20 = 5×x → 60 = 5x → x = 12

Example 3: Mix 2 liters of concentrate for every 5 liters of water. How much concentrate for 20 L water?

  • 2:5 = x:20 → x = (2×20)/5 = 8
  • Answer: 8 L concentrate

6. Practice Questions

  1. Simplify 18:24.
  2. If 4:7 = x:21, find x.
  3. Divide ₹4800 among A, B, C in ratio 3:5:7.
  4. If a:b = 2:3 and b:c = 5:10, find a:c.
  5. Are 12:18 = 28:56? Justify.

7. Solutions

  1. 18:24 = (÷6) = 3:4
  2. 4×21 = 7×x → x = 12
  3. Total parts = 3+5+7 = 15 → A = (3/15×4800)=960; B = 1600; C = 2240
  4. Make b common: 2:3 = 10:15 & 5:10 = 15:30 → a:c = 10:30 = 1:3
  5. 12/18 = 2/3; 28/56 = 1/2 → Not equal, so not proportional.

8. Exam Tips

  • Always simplify ratios before use; find HCF.
  • Identify whether direct, inverse, or continued proportion applies.
  • Apply properties (invertendo, componendo) to avoid cross-multiplying every time.
  • Use LCM/HCF methods to combine chained ratios across multiple quantities.
  • Practice with real-world problems: recipes, sharing, speed-time relationships.
