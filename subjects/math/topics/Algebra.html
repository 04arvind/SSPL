Algebra

1. What is Algebra?

Algebra is a branch of mathematics that deals with symbols and rules for manipulating those symbols. In algebra, letters like x, y, and z are used to represent unknown quantities. Algebra allows you to create mathematical models for real-world situations and solve problems by forming equations.

2. Linear Equations

A linear equation is an algebraic equation in which the highest power of the variable is 1. It represents a straight line when graphed. These equations are of great importance in competitive exams, especially in the arithmetic and algebra sections.

Standard Form

  • General Form: ax + b = c
  • Where:
    • a ≠ 0 (coefficient of x)
    • b is a constant
    • c is the constant on the RHS

3. Example 1: Solve Linear Equation

Solve: 2x + 5 = 11.

  • Subtract 5: 2x = 6.
  • Divide by 2: x = 3.
  • Answer: x = 3.

4. Quadratic Equations

  • Form: ax² + bx + c = 0.
  • Solve using factoring or quadratic formula: x = [-b ± √(b² – 4ac)] / (2a).

5. Example 2: Solve Quadratic Equation

Solve: x² – 5x + 6 = 0.

  • Factorize: (x – 2)(x – 3) = 0.
  • Solutions: x = 2, x = 3.
  • Answer: x = 2, 3.

6. Example 3: Quadratic Formula

Solve: 2x² – 4x – 6 = 0.

  • a = 2, b = -4, c = -6.
  • Discriminant = (-4)² – 4 × 2 × (-6) = 64.
  • x = [4 ± √64] / 4 = 3, -1.
  • Answer: x = 3, -1.

7. Practice Problems

  1. Solve: 3x – 7 = 8.
  2. Solve: x² – 4x – 5 = 0.
  3. Solve: 5x + 3 = 2x + 12.
  4. Solve: 2x² + 3x – 2 = 0.
  5. If 2x + y = 7 and x – y = 1, find x and y.

8. Solutions

  1. x = 5.
  2. x = 5, -1.
  3. x = 3.
  4. x = 1/2, -2.
  5. x = 8/3, y = 5/3.

9. Tips

  • Check solutions by substituting back.
  • Practice factoring quadratics.
  • Use elimination or substitution for simultaneous equations.
